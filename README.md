Setup & Usage

To run any Python script:

python3 script.py

To compile and execute C files:

gcc -Wall -Werror -Wextra -pedantic file.c -o output && ./output

For SQL queries:

mysql -u root -p < script.sql

For JavaScript files:

node script.js
